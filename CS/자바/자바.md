# 자바(JAVA)란?
- 실행환경(JRE) + 개발도구(JDK) +라이브러리(API)
- 모던 프로그래밍 언어
- 객체 지향 언어
- 자동 메모리 관리(가비지 컬렉터)
- 멀티 쓰레드를 지원
- <span style="color:pink">운영체제에 독립적</span>([[자바 가상 머신(JVM)]])
- [[자바 런타임 환경(JRE)]]

# 자바의 역사
-  운영체제에 독립적인 [[Oak]]에서 출발
-  [[서블릿(Servlet)]]과 [[JSP]]가 많이 사용됨

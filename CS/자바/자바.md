# 자바(JAVA)란?
- [[자바 런타임 환경(JRE)]] + [[JDK(자바개발도구)]] +라이브러리(API)
- 모던 프로그래밍 언어
- 객체 지향 언어
- 자동 메모리 관리(가비지 컬렉터)
- 멀티 쓰레드를 지원
- <span style="color:pink">운영체제에 독립적</span>([[자바 가상 머신(JVM)]])
- [[자바 런타임 환경(JRE)]]

# 자바의 역사
-  운영체제에 독립적인 [[Oak]]에서 출발
-  [[서블릿(Servlet)]]과 [[JSP]]가 많이 사용됨

# 자바언어의 특징
- [1] 운영체제에 독립적
- [2] 객체지향언어
- [3] 비교적 배우기 쉬움
- [4]  자동 메모리 관리
- [5] 네트워크와 분산처리 지원
- [7] 동적 로딩 지원
- [6] 멀티쓰레드 지원

# JVM(Java Virtual Machin)
- 자바를 실행하기 위한 가상 컴퓨터
- 자바로 작성된 애플리케이션은 (JVM) 에서만 실행됨
